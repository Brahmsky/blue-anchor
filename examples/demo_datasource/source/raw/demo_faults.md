# Demo Ship Maintenance Notes

## Example Pump A Fails To Start

Symptom: Pump A does not start after the operator presses the start button.

Possible causes:

- The local power switch is off.
- The control fuse is open.
- The start button contact is loose.

Recommended checks:

1. Verify incoming power at the local control box.
2. Check the control fuse and replace it if it is open.
3. Inspect the start button wiring and tighten loose terminals.

## Example Cooling Fan B Overheats

Symptom: Cooling fan B runs with abnormal noise and the motor casing becomes hot.

Possible causes:

- The inlet screen is blocked by dust.
- The bearing lacks lubrication.
- The fan belt tension is too high.

Recommended checks:

1. Stop the fan and clean the inlet screen.
2. Rotate the shaft by hand and check for bearing resistance.
3. Adjust belt tension according to the maintenance manual.
